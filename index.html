Facebook Dating: A UX Research Case Study

Social Media platforms itself is still a relatively new phenomenon. The platform ways in which people have been using the Internet to interact with one another is continually changing to promote connectivity now more than ever. One part of social media, in particular, that has been drawing people together in a unique way, is online dating.

This case study will be focusing on exploring the online dating experience with Facebook’s new feature embedded in the official Facebook application. Using the existing matchmaker applications such as Tinder, Bumble, and Hinge to compare them with Facebook Dating. Furthermore, I will also conduct testing and identifying problems through the conceptual design and mapping between the controls and the system display.

This case study is still in the process of completion.

Easy Animation

This project was a part of our academic course – Object-Oriented Design. This easy animation project focused on creating an animator that had multiple views which rendered the animation by utilizing JAVA Object-Oriented Design principle. Through collaboration, my teammate and I also included customizable options which allowed the user to manipulate the speed, start/stop point, and other aspects of the animation.
