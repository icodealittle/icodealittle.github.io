Easy Animation

This project was a part of our academic course – Object-Oriented Design. This easy animation project focused on creating an animator that had multiple views which rendered the animation by utilizing JAVA Object-Oriented Design principle. Through collaboration, my teammate and I also included customizable options which allowed the user to manipulate the speed, start/stop point, and other aspects of the animation. Due to my agreement with the institute, the university requested students not to set the assignment codes publicly. However, I’m happy to send the file to whomever upon any request.

Sudoku

This puzzle game is designed for a single player while a logical base method to win the entire. One rule that the player must follow: no repeats are allowed in any row, column, or block. Developed an extensible GUI game Python application that applying object-oriented design principles and utilizing the backtracking algorithms. The project also includes a graphical GUI as well as a text-based version.
